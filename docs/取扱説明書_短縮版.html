サッカー練習管理アプリ 取扱説明書

配布用・短縮版

1. このアプリでできること

  • 練習日の作成・管理
  • 練習メニューのストック管理（閲覧・編集）
  • 各チームのカテゴリー割り当て（選手移動）
  • グラウンド枠ごとの練習配置
  • 試合予定（TM/公式戦）の記録
  • 各メニューの総評・留意点の記録

2. 基本の使い方（1日の流れ）

  1. 日付を選ぶ
  2. グラウンド割りを設定（時間・エリア・チーム）
  3. メニュー作成で、ストックから枠へ配置
  4. カテゴリー割り当てで選手をチーム別に整理
  5. 必要に応じて試合情報を入力
  6. 練習後に総評・留意点を記入

3. 主要機能の操作

練習メニュー

  • ストック一覧のカードを開くと、ルール / コーチングポイントを確認可能
  • 必要に応じてその場で編集可能（タイトル・内容・タグ等）

選手管理

  • 選手追加：単体追加 / 一括追加（改行区切り）
  • 選手タグには学年表示あり
  • カテゴリー画面で移動・一括移動が可能

試合登録

  • まず「練習 / 試合」を選択
  • 試合選択時のみ記入：対戦相手 / 種別（TM・公式戦）/ 大会名 / 場所 / 時間

4. スマホ運用のポイント

  • 縦画面前提で操作可能
  • 一部画面は横スクロールで全項目を確認
  • ドラッグしづらい場合は、タップ操作や入力方式で代替可能

5. よくある質問（FAQ）

Q. ドラッグでずれる / うまく置けない
A. 一度指を離して再操作してください。カテゴリ移動は一括移動でも対応できます。

Q. 追加した内容が見当たらない
A. 日付・チームタブ・カテゴリフィルタを確認してください。

Q. 画面が真っ白になる
A. 通信状態と環境変数（Supabase設定）を確認し、再読み込みしてください。

6. 運用ルール（推奨）

  • メニュー名は短く統一（例：3vs2 トランジション）
  • コーチングポイントは1〜3行で簡潔に
  • 試合情報は「相手名・種別・場所・時間」を必須入力
  • 練習後、総評は当日中に記入
